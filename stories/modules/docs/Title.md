
# Title Module

## Example
```jsx
{
  name: "title",
  type: "title",
  title: "Article title",
}
```

## Title properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | 'title'        | true     | -       | module type |
| *name*       | string         | true     | -       | module name |
| *title*      | string         | true     | -       | title text  |
