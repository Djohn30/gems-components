
# Checkbox Module

## Example
```jsx
{
  name: "checkbox",
  type: "checkbox",
  label: "select me",
}
```

## Checkbox properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | 'checkbox'     | true     | -       | module type |
| *name*       | string         | true     | -       | module name |
| *label*      | string         | -        | -       | Checkbox label |
