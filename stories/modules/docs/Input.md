
# Input Module

## Example
```jsx
{
  name: 'text',
  type: 'text',
  placeholder: 'Simple text module',
}
```

## Input properties

| Property       | type           | Required | Default | description |
| -------------- | -------------- | -------- | ------- | ----------- |
| *type*         | 'text' 'number' 'email' 'password'     | true     | -       | module type  |
| *name*         | string         | true     | -       | module name  |
| *placeholder*  | string         | -        | -       | input placeholder  |
