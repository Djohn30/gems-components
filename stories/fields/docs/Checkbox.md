
# Checkbox Field

## Example
```jsx
{
  name: "checkbox",
  type: "checkbox",
  label: "select me",
}
```

## Field properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | 'checkbox'     | true     | -       | field type  |
| *name*       | string         | true     | -       | field name  |
| *label*      | string         | -        | -       | Checkbox label |
