
# Title Field

## Example
```jsx
{
  name: "title",
  type: "title",
  title: "Article title",
}
```

## Field properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | 'title'        | true     | -       | field type  |
| *name*       | string         | true     | -       | field name  |
| *title*      | string         | true     | -       | title text  |
