
# Submit Field

## Example
```jsx
{
  name: 'submit',
  type: 'submit',
  caption: 'Next',
  justify: 'left',
}
```

## Field properties

| Property  | type     | Required | Default | description |
| --------- | -------- | -------- | ------- | ----------- |
| *type*    | 'submit' | true     | -       | field type  |
| *name*    | string   | true     | -       | field name  |
| *caption* | string   | true     | -       | button text   |
| *justify* | 'left', 'right', 'center', 'between'  | -        | 'center'       | horizontal alignment |
